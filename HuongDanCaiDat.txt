1. Cài đặt NodeJs (phiên bản LTS 14.19 trở lên)

2. Cài đặt npm (phiên bản 5.2 trở lên)  

3. Cài đặt PostgreSQL (phiên bản 14 trở lên): https://www.postgresql.org/download/
        Tạo một user và một database trong PostgreSQL
        
        3.1. sudo -u postgres psql
        3.2. CREATE USER <YOUR_USERNAME> WITH PASSWORD <YOUR_PASSWORD>;
        3.3. CREATE DATABASE <YOUR_DB_NAME>;
        3.4. GRANT ALL PREVILLEDGES ON DATABASE <YOUR_DB_NAME> TO <YOUR_USERNAME>;

4. Cài đặt Redis (phiên bản 6.2.6 trở lên)

5. Cài đặt TypeORM (phiên bản 0.2.41 trở lên): 
        npm install -g typeorm

6. Chạy lệnh dưới đây để thực hiện kiểm tra và khởi tạo dữ liệu: 
        npm run init

        Nhập các thông tin yêu cầu (đã tạo ở trên)

7. Chạy unit test:
        npm run test:unit
        
7. Chạy chương trình bằng lệnh:
        npm run start
